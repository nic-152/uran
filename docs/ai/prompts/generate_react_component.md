# Prompt Template: generate_react_component

## Цель
Сгенерировать React-компонент, который работает с backend API проекта `uran`.

## Шаблон
Создай React-компонент `<ComponentName>` для страницы `<PageName>`.

Требования:
- TypeScript, функциональный компонент, хуки React.
- Все данные получать из `/api/...`, не из localStorage (кроме UI-настроек).
- Обрабатывать loading/error/empty состояния.
- Ошибки API показывать пользователю понятным текстом.
- Не использовать хардкод бизнес-данных в компоненте.
- Поддержать адаптивность (desktop + mobile).
- Не нарушать существующий визуальный стиль.

Выведи:
1. Код компонента.
2. Нужные типы/интерфейсы.
3. Как подключить компонент в существующий экран.
4. Какие endpoint'ы ожидаются от backend.
