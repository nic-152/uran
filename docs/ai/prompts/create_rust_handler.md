# Prompt Template: create_rust_handler

## Цель
Сгенерировать новый handler для Axum с использованием `sqlx` и корректной обработкой ошибок.

## Шаблон
Ты работаешь в монорепозитории `uran`.

Сделай handler для endpoint: `<METHOD> <PATH>`.

Требования:
- Используй `axum` + `sqlx` + PostgreSQL.
- Не используй `unwrap`/`expect`.
- Ошибки возвращай в JSON формате `{ "error": "..." }`.
- Валидацию входных данных выполни до запроса в БД.
- Добавь/обнови DTO-структуры `serde`.
- Если меняется модель данных, обнови `docs/ai/02-database.md`.
- Обнови маршрутизацию в `backend/src/main.rs` или соответствующем роутере.
- Приложи минимальный smoke-тест запроса.

Выведи:
1. Список изменённых файлов.
2. Полный patch.
3. Короткий пример curl-запроса.
